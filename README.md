# Sales and Products Management System

A Python and SQLite based Sales and Products Management System built using DBMS concepts.

## Features
- Product management (Add, Update, Delete, View)
- Sales transaction recording
- Automatic inventory updates
- SQLite relational database
- Menu-driven console interface

## Technologies Used
- Python
- SQLite
- SQL (CRUD operations)
- DBMS concepts

## How to Run
```bash
python sales_products_management.py
